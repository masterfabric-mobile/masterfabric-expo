# Application Core Views

This section provides detailed documentation for all major screens of the Voice AI application, ordered by a typical user flow.

### Core Views

- **[0. Splash View](./00-splash-view.md)**: The initial loading and initialization screen.
- **[1. Onboarding View](./01-onboarding-view.md)**: The welcome/auth screen for new users.
- **[2. Dashboard View](./02-dashboard-view.md)**: The main entry point after login.
- **[3. Recording View](./03-recording-view.md)**: The active screen for voice recording.
- **[4. Summary View](./04-summary-view.md)**: Displays the details of a single recording.
- **[5. History View](./05-history-view.md)**: Lists all past recordings.
- **[6. Tasks View](./06-tasks-view.md)**: Manages all tasks generated from summaries.
- **[7. Settings View](./07-settings-view.md)**: The screen for account and app management.
