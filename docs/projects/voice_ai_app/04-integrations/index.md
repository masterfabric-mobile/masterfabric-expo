# 4. Integrations

This section outlines the third-party services integrated into the Voice AI application. Each service is abstracted into its own module for clean, maintainable code.

For detailed information on each integration, see the following documents:

- **[Authentication](./authentication.md)**: User management with Supabase.
- **[Analytics](./analytics.md)**: User engagement tracking with Google Analytics.
- **[Push Notifications](./push-notifications.md)**: Notification services via OneSignal.
- **[Payments](./payments.md)**: Subscription payment processing with Stripe and Iyzico.
- **[Advertisements](./advertisements.md)**: Monetization using Google AdMob and Meta Audience Network.
