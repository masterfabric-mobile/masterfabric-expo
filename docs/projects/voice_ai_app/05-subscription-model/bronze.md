# 🥉 Bronze (Free)

This is the default plan for all new users. It is designed to let users experience the core functionality of the app with certain limitations.

- **Cost**: Free
- **Features**:
  - Limited number of voice recordings per month (e.g., 5 recordings).
  - Basic AI summarization quality.
  - Standard task management features.
- **Monetization**: This tier is supported by advertisements from Google AdMob or Meta Audience Network.
