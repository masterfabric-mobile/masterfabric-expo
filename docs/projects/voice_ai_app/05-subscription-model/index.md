# 5. Subscription Model

The application uses a tiered subscription model to grant users access to features. The core logic for managing this is handled by `SubscriptionContext.tsx`.

### Subscription Tiers

- **[🥉 Bronze (Free)](./bronze.md)**: The entry-level plan for all users.
- **[🥇 Gold (Standard)](./gold.md)**: For regular users needing more features.
- **[💎 Diamond (Premium)](./diamond.md)**: For power users who require unlimited access.
