# 1. Prerequisites

Before you begin, ensure you have the following software installed on your system.

- **[Node.js](https://nodejs.org/)**: The LTS (Long-Term Support) version is highly recommended for compatibility.
- **[Git](https://git-scm.com/)**: For version control and cloning the repository.
- **[Expo CLI](https://docs.expo.dev/get-started/installation/)**: The command-line tool for Expo.
  ```bash
  npm install -g expo-cli
  ```
- **Emulation Software**:
  - For iOS: **Xcode** (from the Mac App Store) to use the iOS Simulator.
  - For Android: **Android Studio** to use the Android Emulator.
