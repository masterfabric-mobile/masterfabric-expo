# 4. Running the App

Once all dependencies are installed and the environment is configured, you can start the Expo development server.

```bash
npm start
```

This command will:
1.  Start the Metro bundler, which compiles your JavaScript code.
2.  Open the Expo Developer Tools in your web browser.

From the Expo Dev Tools, you can run the application on:
- **An Emulator**: Press `i` to open the app in the iOS Simulator or `a` to open it in the Android Emulator.
- **A Physical Device**: Scan the QR code shown in the terminal or browser with the **Expo Go** app on your iOS or Android phone.
