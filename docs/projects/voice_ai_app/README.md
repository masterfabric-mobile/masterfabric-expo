# Voice AI Application Documentation

This documentation provides a technical overview of the **Voice AI Application**, a mobile app designed for voice recording, AI-powered summarization, and task note creation.

The project is built on the MasterFabric Expo ecosystem, leveraging `@masterfabric-expo/core` for a scalable and consistent architecture.
